What is R?

R programming language

  • “language and environment for statistical computing and graphics” - https://www.r-project.org/about.html
  • free and open source
  • import data, data exploration, statistical analysis, graphical representation, report creation

R

RStudio

Downloading R

Navigate to https://cran.r-project.org/

Downloading R Studio

Navigate to https://posit.co/download/rstudio-desktop/

R Studio

Writing code

Maths

You can do simple math in the console.

1 + 1
[1] 2

You can use pi as well as other more complex arithmetic.

4*pi
[1] 12.56637
10^2
[1] 100

Assigning variables

By convention, variables are assigned using <-

x <- 5
x
[1] 5
x*2
[1] 10
Navigating history of the console

You can press arrow keys ↑ and ↓ to go backward and forward through the history of code that was executed. This is useful for quickly going back to fix an error in code.

Types of variables

Creating a script

A script is a file containing text, written in the R programming language, which can be executed in R.
File -> New File -> R Script

  • Stay organised (logical flow)
  • Repeat the same analysis with new data
  • Share code with others
Executing your script

Click on a line in your script and press Ctrl + ↵ Return to execute your script line-by-line.

Writing code

Creating vectors

A vector is simply a list of items that are all the same type.
The function c() combines values to a vector (list).
Functions follow the pattern function_name(argument/s).

x <- c(1,4,5.8) # I am assigning my favourite numbers to the variable "x"
x*2             # Multiply each number by 2
[1]  2.0  8.0 11.6
Commenting on code

Everything to the right of the hashtag (#) is considered a “comment” and is not interpreted as R code.

Writing code

What’s my variable?

We can find out the structure of a variable or dataframe is is by using the function str()

x <- c(1,4,5.8) # I am assigning my favorite numbers to the variable "x"
str(x)
 num [1:3] 1 4 5.8
y <- "fish"
str(y)
 chr "fish"

Base R and packages

Base R and packages

  • R comes with various functions, packages, and datasets “baked in”

  • c(), str(), sqrt(), log(), etc.

  • Built-in datasets

  • You can get help within R Studio by typing a question mark before a function, package, or dataset (i.e. ?str())

Base R and packages

Packages

  • packages are created by R users to expand functionality of R

  • statistical packages, graphical packages, unique datasets, etc.

Base R and packages

We will use many datasets from the datasets package built into R.

One such dataset is called mtcars.
